# Next.js API Routes

Чтобы настроить API, нам необходимо создать папку под названием "api"
внутри папки "Pages".
Там мы можем точно также создавать пути, которые будут соответсвовать
нашей файловой структуре.

Example:

pages |
      |  
      | api |
            |      
            | feedback.ts - domain.com/api/feedback.ts

Внутри этого файла, мы создаём функцию, под названием "handler",
которая принимает два объекта "req", "res".
С которыми мы можем проводить манипуляции.

<p style="color:red">Код написанный в папке API никогда не попадёт к клиенту.</p>

## Dynamic routes
Достигаются таким же путём как и обычные страницы с динамической переменной.
